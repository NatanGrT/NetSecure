Qu'est-ce qu'une Adresse IP ?

Une adresse IP (Internet Protocol) est un identifiant unique pour chaque appareil connecté à un réseau. Elle permet de communiquer et de localiser les appareils sur Internet ou sur un réseau local.

Types d'Adresses IP

  • Adresse IP publique : Identifie un appareil sur Internet.
  • Adresse IP privée : Utilisée pour identifier des appareils au sein d'un réseau interne.
  • Adresse IP dynamique : Assignée automatiquement par le DHCP et peut changer périodiquement.
  • Adresse IP statique : Attribuée manuellement et reste fixe.

Configuration Automatique (DHCP)

La configuration DHCP (Dynamic Host Configuration Protocol) permet d’attribuer automatiquement des adresses IP aux appareils connectés au réseau. C’est un choix populaire pour les réseaux domestiques et d’entreprise, car il réduit la complexité de gestion.

Configuration Manuelle (IP Statique)

Une adresse IP statique est configurée manuellement et ne change pas. Elle est souvent utilisée pour les serveurs, les imprimantes réseau, ou d’autres appareils qui doivent avoir une adresse fixe.

Pour configurer une adresse IP statique, les paramètres suivants sont nécessaires :

  • Adresse IP : L’adresse spécifique assignée à l’appareil.
  • Masque de sous-réseau : Utilisé pour déterminer la partie réseau de l’adresse IP.
  • Passerelle par défaut : Adresse IP de l'appareil qui permet la connexion avec d'autres réseaux.
  • Serveur DNS : Serveurs permettant la résolution des noms de domaine en adresses IP.

Exemple de Configuration d'une IP Statique

Pour configurer une IP statique sur un réseau local :

  1. Accédez aux paramètres réseau de votre appareil.
  2. Sélectionnez "Manuel" ou "Statique" dans les options IP.
  3. Entrez l'adresse IP souhaitée (ex. 192.168.1.10).
  4. Ajoutez le masque de sous-réseau (ex. 255.255.255.0).
  5. Ajoutez la passerelle par défaut (ex. 192.168.1.1).
  6. Indiquez les serveurs DNS (ex. 8.8.8.8 pour Google DNS).
