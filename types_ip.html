Adresse IP Publique

Une adresse IP publique est attribuée à un appareil pour communiquer sur Internet. Elle est unique à l’échelle mondiale et peut être utilisée pour identifier le réseau d'un utilisateur.

Adresse IP Privée

Une adresse IP privée est utilisée au sein d'un réseau local (LAN). Elle est attribuée aux appareils sur un réseau domestique ou d’entreprise et n'est pas accessible depuis Internet directement.

Adresse IP Statique

Une adresse IP statique ne change pas au fil du temps. Elle est souvent utilisée pour les serveurs et autres dispositifs nécessitant une adresse IP fixe, comme les imprimantes réseau et les caméras de sécurité.

Adresse IP Dynamique

Une adresse IP dynamique est attribuée temporairement à un appareil à chaque connexion au réseau. Elle est souvent assignée par le protocole DHCP (Dynamic Host Configuration Protocol).

Adresse IPv4

IPv4 est le format d'adresse IP le plus courant, composé de quatre nombres séparés par des points (par exemple, 192.168.1.1). Il utilise un espace d'adressage de 32 bits, permettant environ 4,3 milliards d'adresses uniques.

Adresse IPv6

IPv6 est le nouveau format d'adresse IP, conçu pour remplacer IPv4. Il utilise un espace d'adressage de 128 bits, permettant un nombre presque illimité d'adresses (environ 340 sextillions), et est représenté en hexadécimal.

Adresse IP de Bouclage (Loopback)

Une adresse IP de bouclage est utilisée pour tester les équipements réseau sur un appareil sans sortir sur le réseau. L’adresse 127.0.0.1 est l’adresse de bouclage standard en IPv4.

Adresse Multicast

Les adresses multicast sont utilisées pour envoyer des données à plusieurs destinataires en une seule transmission. Elles commencent généralement par 224.0.0.0 à 239.255.255.255 en IPv4.
