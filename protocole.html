Protocoles de Transport

  • TCP (Transmission Control Protocol): Protocole de transport fiable, utilisé pour des connexions stables.
  • UDP (User Datagram Protocol): Protocole de transport rapide, utilisé pour les transmissions sans garantie de livraison.

Protocoles Internet

  • IP (Internet Protocol): Fournit des adresses pour les appareils connectés et la livraison de paquets.
  • ICMP (Internet Control Message Protocol): Utilisé pour les diagnostics réseau, tel que le ping.
  • ARP (Address Resolution Protocol): Associe les adresses IP aux adresses MAC pour la communication sur le réseau local.

Protocoles Web

  • HTTP (HyperText Transfer Protocol): Protocole utilisé pour le transfert de données sur le Web.
  • HTTPS (HTTP Secure): Version sécurisée de HTTP, qui chiffre les données via SSL/TLS.

Protocoles de Messagerie

  • SMTP (Simple Mail Transfer Protocol): Protocole d'envoi de courriels.
  • POP3 (Post Office Protocol 3): Récupération des courriels depuis un serveur.
  • IMAP (Internet Message Access Protocol): Récupération et gestion des courriels stockés sur le serveur.

Protocoles de Sécurité

  • SSL/TLS (Secure Sockets Layer / Transport Layer Security): Protocole de sécurisation des échanges via le chiffrement.
  • SSH (Secure Shell): Protocole pour l'accès sécurisé à distance.
  • IPsec (Internet Protocol Security): Suite de protocoles sécurisant les échanges au niveau IP.

Protocoles de Réseau Local

  • DHCP (Dynamic Host Configuration Protocol): Attribution automatique d'adresses IP.
  • DNS (Domain Name System): Résolution des noms de domaine en adresses IP.
